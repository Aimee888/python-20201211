# 2048 小游戏
## game_2048.py
4 * 4的小方格
游戏规则：  
~~~
1. 先随机在方格中产生2个数，2或者4（产生2的几率90%）。  
2. 然后可以按（【上|下|左|右】或【W|S|A|D】）方向键，使数字朝一个方向移动。  
3. 当格子上出现2048数字时，即表示游戏通关。  
4. 若方格中占满了数字，并且无法移动，则游戏失败。
5. 排序方式：若是左右移动，取每行。上下移动，取每列。  
若向左移动，则从右向左扫描，碰到相同的数字就相加给左边那位，右边置零。
若左边那位是0，就将右边那位移到左边，右边那位置零。一轮之后，刷新列表，
继续从右向左扫描，直到不能移动为止

举例：
[2 2 2 2] 向左 [8 0 0 0]  
[2 2 2 0] 向左 [2 4 0 0]  
[2 2 0 0] 向左 [4 0 0 0]  
[2 0 0 0] 向左 [2 0 0 0]  
[2 2 2 4] 向左 [2 8 0 0]  
[4 4 4 0] 向左 [4 8 0 0]  
~~~

## game_2048_update.py
4 * 4的小方格

~~~

2. 然后可以按（【上|下|左|右】或【W|S|A|D】）方向键，使数字朝一个方向移动。  
3. 当格子上出现2048数字时，即表示游戏通关。  
4. 若方格中占满了数字，并且无法移动，则游戏失败。
5. 排序方式：若是左右移动，取每行。上下移动，取每列。

退出扫描，剩下的依次移到左边
举例：
[2 2 2 2] 向左 [4 2 2 0]  
[2 2 2 0] 向左 [4 2 0 0]  
[2 2 0 0] 向左 [4 0 0 0]  
[2 0 0 0] 向左 [2 0 0 0]  
[2 2 2 4] 向左 [4 2 4 0]  
[4 4 4 0] 向左 [8 4 0 0]  
~~~
